# Git-Commands
Here are some git commands which are useful.

1. **git init**  
    This command initializes a new Git repository in the current directory.
2. **git clone [repository]**  
    This command clones an existing Git repository from a remote source, such as GitHub or GitLab, to your local machine.
3. **git log**  
    This command displays a log of the commit history for the repository, including the author, date, and message for each commit.
4. **git diff**  
    This command displays the differences between two commits, branches, or files.
5. **git fetch**  
    This command retrieves new information from a remote repository without merging the changes into the local branches.
6. **git reset [file]**  
    This command discards changes for a specific file or for the entire repository, allowing to revert to a previous version.
7. **git status**  
    This command displays the current status of the repository, including changes that have been staged or committed, and changes that have not yet been staged.
8. **git add [file]**  
    This command stages changes for a file or group of files to be included in the next commit.
9. **git commit -m [message]**  
    This command creates a new commit with the staged changes and includes a message to describe the changes.
10. **git pull [remote] [branch]**  
    This command fetches and merges changes from a remote repository into a local branch.
